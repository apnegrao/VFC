O Game utiliza a biblioteca Qt Jambi da trolltech.

Para compilar/correr, fazer download da qt jambi a partir de:
	http://trolltech.com/downloads/opensource
e colocar os jars:
	qtjambi-versão.jar		[ necessário para compilar e correr ]
	qtjambi-plataforma-versão.jar	[ necessário só para correr]
na pasta lib/ .

That is all.
Testado com Qt Jambi 4.4 Preview.
